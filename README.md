

**Stellar Space** is a bite-sized game developed in exactly **one week** as part of a game jam-style sprint. Built entirely around the theme of **Randomness**.


[![Play on itch.io](https://img.shields.io/badge/Play_on-itch.io-FA5C5C?style=for-the-badge&logo=itch.io)](https://newkerim.itch.io/stellar-space)

Built under a strict 7-day time limit, this project was driven by three main objectives: learning the Unity engine, collaborating effectively as a development team, and successfully delivering a working version of the game we envisioned.
